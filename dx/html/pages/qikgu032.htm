5.6 Using the Header File to Import Data

Once you have created a Data Prompter header file, the next step is to use that file to import the data it describes. The basic procedure follows:

  1. Place an Import tool icon on the VPE canvas (see "Selecting tools and placing icons").
  2. Double click on the icon to invoke the configuration dialog box (see Figure 19).
  3. The following information is required:
    1. The (path) name of the header file to be imported (in the first text field of the Value column (at right). If the Import module is part of a network displayed in the VPE, the path name appears automatically.
    2. Set format (the third field of the column)
    3. Set format (the third field of the column) to "general" unless the header file already has that as its extension.

    Figure 19. Import Configuration Dialog Box.  



    Notes:

    1. Header files created with the Data Prompter are given this extension automatically when they are saved with Save As... (Step 10 of Example 1 in To save the header file).

    2. The variable parameter can be used to import a subset of the variables specified by the field keyword statement. For the General Array format, by default, all variables are imported.
The specified data file will now be imported when any visualization program containing this Import module is run.
