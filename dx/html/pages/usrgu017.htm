Backward Incompatibilities

There are a few backward incompatibilities with previous versions of Data Explorer. You can click on a module name to go directly to that module's description if applicable.

WriteImage and ReadImage

gif format has been removed due to patent issues with the LZW compression algorithm owned by UNISYS. In order to read and write GIF images, use the added ImageMagick supported format, which doesn't have this problem.

New Accelerator Keys

It is possible that some versions may use the new accelerator key assignments. For those that have used DX for a while, the change may be akward and they may wish to recompile DX with the old keymappings.
